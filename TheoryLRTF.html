Longest Remaining Time

The full form of LRTF is Longest Remaining Time First. It is also known as LJF preemptive scheduling. In this method, the process will be allocated to the task, which is farthest to its completion. This method prevents a newer ready state process from holding the completion of an older process.


Characteristics of LRTF scheduling method:


• This method is mostly applied in batch environments where long jobs are required to be given preference.
• This is not an ideal method to implement it in a shared system where the required CPU time is unknown.
• Associate with each process as the length of its next CPU burst. So that operating system uses these lengths, which helps to schedule the process with the longest possible time.
